# Python - Exceptions

Bu layihə **Holberton School** kurikulumunun bir hissəsidir və Python-da xətaların idarə edilməsini (**Exceptions**) əhatə edir. Layihə çərçivəsində `try`, `except` və `finally` bloklarından istifadə edərək proqramın xətalar zamanı çökməsinin qarşısını almağı öyrənirik.

## Layihə Tapşırıqları

| Fayl | Təsvir |
| --- | --- |
| **0-safe_print_list.py** | Siyahının ilk x elementini təhlükəsiz şəkildə çap edir. |
| **1-safe_print_integer.py** | Verilən dəyəri yalnız tam ədəd (`int`) olduğu halda çap edir. |
| **2-safe_print_list_integers.py** | Siyahıdakı yalnız tam ədədləri çap edir. |
| **3-safe_print_division.py** | İki tam ədədin bölünməsini həyata keçirir və nəticəni çap edir. |
| **4-list_division.py** | İki siyahını element-element bölür və yeni siyahı yaradır. |
| **5-raise_exception.py** | `TypeError` xətası çıxaran funksiya. |
| **6-raise_exception_msg.py** | Mesaj ilə birlikdə `NameError` xətası çıxaran funksiya. |

## Tələblər

- Əməliyyat sistemi: **Ubuntu 20.04 LTS**
- Dil: **Python 3.8.5**
- Kodlar **Pycodestyle** (v2.8.*) standartlarına uyğun yazılmışdır.
