# Python - Network #0

Bu layihədə Bash skriptləri və `curl` istifadə edərək HTTP sorğularının (GET, POST, DELETE, OPTIONS) göndərilməsini və cavabların (headers, body size) emalını öyrənirik.

## Fayllar və Tapşırıqlar
- **0-body_size.sh**: Cavabın body ölçüsünü baytla göstərir.
- **1-body.sh**: GET sorğusu göndərir və body-ni çap edir.
- **2-delete.sh**: DELETE sorğusu göndərir.
- **3-methods.sh**: Serverin dəstəklədiyi HTTP metodlarını göstərir.
- **4-header.sh**: Xüsusi header ilə sorğu göndərir.
- **5-post_params.sh**: POST parametrləri ilə sorğu göndərir.

## Tələblər
- Bütün skriptlər tam 3 sətirdən ibarətdir.
- Ubuntu 20.04 LTS mühitində test edilib.
